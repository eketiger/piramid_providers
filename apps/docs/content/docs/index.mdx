# Piramid Providers

Plataforma operativa para proveedores de la red de Piramid. Administrá licitaciones, órdenes,
visitas, reportes y facturación desde una sola interfaz.

## Para quién es

- Técnicos de electro línea blanca y climatización
- Talleres mecánicos de siniestros
- Prestadores médicos a domicilio
- Logística de repuestos y courier urgente

## Cómo empezar

1. [Registrate y completá el onboarding](/docs/onboarding)
2. [Cotizá tu primera licitación](/docs/licitaciones)
3. [Agendá visitas y subí reportes](/docs/visitas-y-reportes)
