# Visitas y reportes

Cada orden tiene seis tabs: Detalle, Actividad, Novedades, Reportes, Facturas, Visitas.

## Visitas

Desde el tab **Visitas** agendás una nueva visita. El cliente recibe un email y un WhatsApp
con la confirmación.

## Reportes

En **Reportes** subís el diagnóstico inicial, fotos del antes/después, el presupuesto y la
factura. Cada archivo se asocia a una **ronda** de revisión. Si el tramitador pide cambios,
se abre una ronda nueva y seguís cargando ahí.

## Facturas

Podés adjuntar la factura A desde el tab **Facturas**. Una vez conformada, el pago se dispara
según el acuerdo con la tramitadora (típicamente a 30 días).
