# Licitaciones

Cuando una aseguradora o tramitadora envía un trabajo, aparece en tu lista de licitaciones
con un SLA de cotización. Cotizá dentro del SLA y vas a aparecer bien rankeado frente a la
competencia.

## Cómo cotizar

1. Abrí el drawer de la licitación desde el listado.
2. Cargá **mano de obra** y **repuestos** por separado — el total se calcula solo.
3. Elegí un **plazo comprometido** (no lo sobreestimes, cuenta para tu score).
4. Enviá. La licitación pasa a estado `cotizada`.

## SLA

El indicador tiene tres estados:

- **Verde** — tenés tiempo de sobra.
- **Naranja** — estás en la ventana de riesgo (≥ 50%).
- **Rojo** — o superaste el 80% o ya venció.

Mantené el verde para el 90% de las cotizaciones y tu score sube.
